# PNGs de ação — Deep Rock Galactic: Survivor

Coloque aqui um arquivo `.png` para cada ação deste jogo (fundo transparente
de preferência). O nome do arquivo precisa ser **exatamente** o `id` da ação,
em minúsculas — não use o nome bonito que aparece na tela do painel.

| Arquivo esperado | Ação | Ícone atual |
|---|---|:---:|
| `spawn_zombie.png` | Spawn Zombie | 🧟 |
| `give_item.png` | Give Any Item | 🔨 |
| `change_health.png` | Change Health | ❤️ |
| `custom.png` | Custom Command | ⚙️ |

Se um arquivo não existir, o painel continua mostrando o ícone emoji no lugar
(nada quebra, só falta a arte).

> Nota: essa lista vem de `ACTION_TYPES` em `js/data.js` (lista genérica,
> compartilhada por todos os jogos "normais"). Se um dia esse jogo ganhar
> ações próprias, atualize esta tabela também.
